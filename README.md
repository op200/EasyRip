# Easy Rip

自用压制脚本

### 获取

在[Actions](https://github.com/op200/EasyRip/actions)中下载最新的exe

或者在[Releases](https://github.com/op200/EasyRip/releases)中下载exe或包

### 使用

运行`easyrip help`查看帮助

### 依赖

Python
```
pip install -U loguru
```

外部
```
ffmpeg.exe
flac.exe
mp4box.exe
mp4fpsmod.exe
mkvpropedit.exe
mkvmerge.exe
```
